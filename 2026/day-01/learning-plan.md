# 90 Days Personal DevOps Learning Plan

## Current Level

I am a Computer Science and Engineering student with beginner-level experience in programming. 
Now I am willing to do a transition into DevOps and Cloud Engineering with no prior hands-on experience.


## My knowledge about DevOps and Cloud Engineering

DevOps is a culture and a set of practices that automates and integrates software development and IT operations to ensure faster delivery of applications.

Cloud engineering is all about building, deploying and managing applications on Cloud platforms like AWS, AZURE, UTHO etc to ensure availability, scalability and cost efficiency.


## Why I am learning DevOps and Cloud

- To achieve a high-paying DevOps engineering role
- To build real world systems beyond just coding
- To become industry ready with modern tools like Docker, Kubernetes, CI/CD, AWS etc
- To gain practical skills that companies actually demand



## 3 clear goals

- I want to gain strong knowledge on Linux, Networking, AWS cloud services
- Build and deploy a full stack application using Docker and CI/CD Pipeline
- Deploy a production-grade application on Kubernetes (EKS or Minikube)


## 3 core DevOps skills I will build

- Linux and System Fundamentals (Commands, Processes, troubleshooting)
- CI/CD Pipelines (Github Actions / Jenkins)
- Containerization and Orchestration (Docker + Kubernetes)
